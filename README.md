# The Collective | PHP Online Shopping Hub

**The Collective** is a fully functional E-commerce platform developed using PHP and MySQL. This project is designed to provide a seamless online shopping experience, featuring a dynamic frontend and a robust Admin Panel. It is an ideal reference for IT students and developers looking to understand the core mechanics of an e-commerce system.

---

## 🚀 Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
* **Backend:** PHP (Core PHP)
* **Database:** MySQL
* **Server Environment:** XAMPP / WAMP / MAMP

---

## 💻 Supported Operating Systems
This project is cross-platform and can run on:
* Windows
* macOS
* Linux

---

## ✨ Key Features

### Client-Side Features:
* **User Management:** Customer registration and secure login system.
* **Shopping Experience:** Advanced Add-to-Cart system and streamlined Checkout process.
* **Product Discovery:** Search and filter products by categories (Top, Mid, and End level).
* **Dynamic Sections:** Separate sections for **Featured**, **Latest**, and **Popular** products (based on view counts).
* **Personal Dashboard:** Customers can view order history and update their profiles/billing addresses.

### Admin Panel Features:
* **Total Control:** Manage products, categories (size, color), and inventory.
* **Order Management:** Track pending and completed orders, and manage shipping details.
* **Website Customization:** Update website branding, social media links, and footer/header settings.
* **Dynamic UI:** Set up image sliders and manage "Services" sections on the home page.
* **Customer Insights:** View registered customers and manage subscribers.

## 🛠️ Installation Guide

Follow these steps to set up the project locally using XAMPP:

1.  **Extract the Files:** Download and extract the project zip file.
2.  **Move to Server:** Copy the main project folder and paste it into your `xampp/htdocs/` directory.
3.  **Start Services:** Open XAMPP Control Panel and start **Apache** and **MySQL**.
4.  **Database Setup:**
    * Open your browser and go to: `http://localhost/phpmyadmin/`
    * Create a new database named: **`ecommerceweb`**
    * Click on the **Import** tab.
    * Click **Choose File** and select the `ecommerceweb.sql` file (found inside the `DATABASE FILE` folder).
    * Click **Go** to import the tables.
5.  **Run the Project:**
    * Open your browser and go to: `http://localhost/TheCollective` 
    *(Note: Replace 'TheCollective' with your actual folder name if different)*
